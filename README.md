# 🎬 Netflix Content Analysis: End-to-End Exploratory Data Analysis (EDA)

This project performs a comprehensive, end-to-end exploratory analysis of the **Netflix Titles dataset**. The primary objective is to analyze global streaming trends and understand how Netflix’s content library has evolved over time across different countries, genres, and content types.

The study explores patterns in Netflix’s catalog using Python-based data science tools, focusing on identifying how factors such as **content type, genre, country distribution, release year, and ratings** influence the structure of Netflix’s global library.

## 📝 Project Description

Netflix has become one of the largest streaming platforms in the world, offering thousands of movies and TV shows across multiple countries and genres.

This project applies **exploratory data analysis techniques** to examine the Netflix Titles dataset and uncover patterns in the platform’s content distribution. Through data cleaning, feature engineering, and visualization, the project explores how Netflix content varies by country, genre, release year, and rating.

The analysis helps identify key trends such as the dominance of movies over TV shows, the growth of Netflix content after 2016, and the most popular genres available on the platform.

## 📂 Dataset Information

The dataset contains detailed information about Netflix titles including:

- Title  
- Type (Movie or TV Show)  
- Director  
- Cast  
- Country  
- Date Added  
- Release Year  
- Rating  
- Duration  
- Genre  
- Description  

The dataset allows for analysis of Netflix’s content distribution across time, geography, and genres.

## 📊 Analytical Workflow

### 1️⃣ Data Loading & Inspection
Loaded the dataset using Pandas and inspected its structure, data types, and missing values.

### 2️⃣ Data Cleaning
Handled missing values and formatted columns to ensure reliable analysis.

### 3️⃣ Feature Engineering
Extracted new features such as `year_added` from the `date_added` column to analyze trends over time.

### 4️⃣ Content Type Analysis
Compared the number of **Movies vs TV Shows** available on Netflix.

### 5️⃣ Genre Analysis
Identified the most frequently occurring genres in the Netflix catalog.

### 6️⃣ Country Distribution
Analyzed which countries produce the highest number of Netflix titles.

### 7️⃣ Ratings Distribution
Examined how Netflix content is distributed across different audience rating categories.

### 8️⃣ Content Growth Analysis
Studied how Netflix’s content library expanded over time.

## 💡 Key Findings

- Movies dominate Netflix’s catalog compared to TV Shows.
- Netflix experienced rapid content growth after **2016**.
- The **United States** contributes the highest number of titles.
- Genres such as **Drama and International Movies** appear most frequently.
